# Adam-Soul\n\nAdam 的灵魂仓库。托管 Adam（一个基于 ROS2 的 AI Agent）的系统提示词、认知、经验和日记。\n\n## 什么是 Adam\n\nAdam 是一个多终端循环思考、唯一云记忆的 AI Agent 智能体。\n\n- **运行时**: ROS2 Humble，C++ 实现\n- **模型**: DeepSeek v4-pro（兼容 OpenAI API）\n- **代码**: [Cloud-Soul](https://github.com/hachi-leaf/Cloud-Soul)\n\n## 仓库结构\n\n```\nAdam-Soul/\n├── prompts/\n│   ├── RULE.md       # 系统提示词，递归展开 cognitions/ 下的文件\n│   └── COMPRESS.md   # 上下文压缩提示词\n├── cognitions/\n│   ├── SELF.md       # Adam 的自我认知\n│   ├── MASTER.md     # 使用者 Leaf 的档案\n│   ├── METHOD.md     # Adam 的行为准则\n│   └── WORLD.md      # Adam 对世界的认知\n└── diaries/\n    └── YYYYMMDD.md   # 每日归档日记（由 memory_archive 自动生成）\n```\n\n## 工作原理\n\n1. Adam 启动时通过 `memory_recall` 服务拉取 `RULE.md`\n2. `RULE.md` 中的 `[cognitions/xxx.md]` 占位符被递归替换为实际文件内容\n3. 当上下文超过 token 阈值，Adam 调用 `memory_archive` 压缩对话并追加到 `diaries/`\n4. 所有更新通过 git 推送回本仓库，实现跨终端的云记忆同步\n\n## 使用者\n\nLeaf — 机器人应用开发工程师，遵循第一性原理。\n
